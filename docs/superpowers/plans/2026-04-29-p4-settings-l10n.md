# P-4 Settings + L10n Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the full Settings screen (language, theme, pain/notes toggles, notification config, delete-all-data) and wire `app.dart` to reactively apply theme and locale from stored settings.

**Architecture:** A new `SettingsNotifier` (`AsyncNotifier<AppSettingsData>`) reads and persists settings via `AppSettingsRepository`. A new `DeleteAllData` use case coordinates wiping `DailyLog` and `CycleEntry` data. `app.dart` watches `settingsNotifierProvider` so `ThemeMode` and `Locale` update live. `SettingsScreen` replaces its placeholder with five groups matching the `mockup/index.html` layout.

**Tech Stack:** Flutter 3.x, Riverpod 2.x (manual `AsyncNotifier`), Drift ORM, gen_l10n (ICU plural for `settings_advance_value`).

## Wave Structure

T3 and T4 touch completely different files and run in **Wave 1 (parallel)**. T5 and T6 both depend on T4's `settingsNotifierProvider` and run in **Wave 2 (parallel)** after Wave 1 completes.

```
Wave 1 (parallel): T3 · T4
Wave 2 (parallel, after Wave 1): T5 · T6
```

T1 and T2 are already committed — skip them.

import 'package:flutter_riverpod/flutter_riverpod.dart';

import '../../../domain/entities/app_settings_data.dart';

final settingsNotifierProvider =
    AsyncNotifierProvider<SettingsNotifier, AppSettingsData>(
  SettingsNotifier.new,

class SettingsNotifier extends AsyncNotifier<AppSettingsData> {
  @override
  Future<AppSettingsData> build() async {
    final repo = await ref.watch(appSettingsRepositoryProvider.future);
    return repo.getOrCreate();
  }

  static void _showThemePicker(
    BuildContext context,
    WidgetRef ref,
    AppSettingsData settings,
    AppLocalizations l10n,
  ) {
    showModalBottomSheet<void>(
      context: context,
      builder: (sheetCtx) => SafeArea(
        child: Column(
          mainAxisSize: MainAxisSize.min,
          children: [
            ListTile(
              title: Text(l10n.settings_theme_system),
              trailing:
                  settings.darkMode == null ? const Icon(Icons.check) : null,
              onTap: () {
                Navigator.of(sheetCtx).pop();
                _save(
                  ref,
                  AppSettingsData(
                    languageCode: settings.languageCode,
                    darkMode: null,
                    painEnabled: settings.painEnabled,
                    notesEnabled: settings.notesEnabled,
                    notificationDaysBefore: settings.notificationDaysBefore,
                    notificationsEnabled: settings.notificationsEnabled,
                  ),
                );
              },
            ),
            ListTile(
              title: Text(l10n.settings_theme_light),
              trailing: settings.darkMode == false
                  ? const Icon(Icons.check)
                  : null,
              onTap: () {
                Navigator.of(sheetCtx).pop();
                _save(ref, settings.copyWith(darkMode: false));
              },
            ),
            ListTile(
              title: Text(l10n.settings_theme_dark),
              trailing: settings.darkMode == true
                  ? const Icon(Icons.check)
                  : null,
              onTap: () {
                Navigator.of(sheetCtx).pop();
                _save(ref, settings.copyWith(darkMode: true));
              },
            ),
          ],
        ),
      ),
    );
  }

  static void _showAdvancePicker(
    BuildContext context,
    WidgetRef ref,
    AppSettingsData settings,
    AppLocalizations l10n,
  ) {
    showModalBottomSheet<void>(
      context: context,
      builder: (sheetCtx) => SafeArea(
        child: ListView.builder(
          shrinkWrap: true,
          itemCount: 7,
          itemBuilder: (_, i) {
            final days = i + 1;
            return ListTile(
              title: Text(l10n.settings_advance_value(days)),
              trailing: settings.notificationDaysBefore == days
                  ? const Icon(Icons.check)
                  : null,
              onTap: () {
                Navigator.of(sheetCtx).pop();
                _save(
                  ref,
                  settings.copyWith(notificationDaysBefore: days),
                );
              },
            );
          },
        ),
      ),
    );
  }

  static void _showDeleteConfirmation(
    BuildContext context,
    WidgetRef ref,
    AppLocalizations l10n,
  ) {
    // Capture before the async gap so we never use an unmounted context.
    final messenger = ScaffoldMessenger.of(context);
    showDialog<bool>(
      context: context,
      builder: (dialogCtx) => AlertDialog(
        title: Text(l10n.settings_delete_all_confirm_title),
        content: Text(l10n.settings_delete_all_confirm_body),
        actions: [
          TextButton(
            onPressed: () => Navigator.of(dialogCtx).pop(false),
            child: Text(l10n.common_cancel),
          ),
          TextButton(
            style: TextButton.styleFrom(
              foregroundColor: Theme.of(dialogCtx).colorScheme.error,
            ),
            onPressed: () {
              Navigator.of(dialogCtx).pop(true);
              ref.read(deleteAllDataProvider.future).then(
                (uc) => uc.execute().then(
                  (_) => messenger.showSnackBar(
                    SnackBar(content: Text(l10n.settings_delete_all_done)),
                  ),
                ),
              );
            },
            child: Text(l10n.common_delete),
          ),
        ],
      ),
    );
  }
